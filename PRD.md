# LifeGamer - 人生游戏化自我提升APP

## 产品需求文档 (PRD)

**版本**: 1.0
**日期**: 2026-06-15
**作者**: Axel

---

## 1. 产品概述

### 1.1 产品愿景

LifeGamer 是一款将人生自我提升游戏化的手机应用，帮助用户通过记录日常生活、追踪情绪变化、设定人生目标，以游戏化的方式激励用户持续成长。

### 1.2 产品定位

- **核心理念**: 人生就是一场游戏，每一天都是升级的机会
- **目标平台**: iOS + Android（跨平台）
- **数据存储**: 本地优先，保护用户隐私
- **差异化**: 游戏化自我提升 + 情绪可视化 + 人生时间管理

### 1.3 产品口号

> "记录今天，升级明天"

---

## 2. 目标用户

### 2.1 用户画像

| 特征 | 描述 |
|------|------|
| 年龄 | 18-35岁 |
| 职业 | 学生、职场新人、自由职业者 |
| 特点 | 有自我提升意愿，喜欢游戏化体验 |
| 痛点 | 缺乏持续动力，难以坚持记录 |
| 需求 | 简单易用的记录工具，可视化的成长反馈 |

### 2.2 使用场景

1. **晨间规划**: 设定今日目标，查看人生进度
2. **随手记录**: 突发灵感、心情变化、重要时刻
3. **晚间回顾**: 记录今日心情，完成游戏任务
4. **周末复盘**: 查看本周/本月情绪趋势
5. **年度回顾**: 总结成长变化，制定新年目标

#### 3.5.5 跨设备同步方案

由于要求"换了手机也能提醒"，提供以下方案：

| 方案 | 优点 | 缺点 | 推荐度 |
|------|------|------|--------|
| iCloud/Google同步 | 原生支持，无需服务器 | 仅限同生态系统 | ⭐⭐⭐ |

- ✅ 日记记录（文字+心情）

**MVP之后迭代：**
- 录音功能

## 8. 风险与挑战

| 风险 | 影响 | 应对策略 |
|------|------|----------|
| 语音转文字准确度 | 用户体验差 | 提供多种方案选择，支持手动编辑 |
| 跨设备数据同步 | 数据丢失 | 优先本地存储，提供导出功能 |
| 用户留存率低 | 产品失败 | 游戏化激励，推送提醒 |
| 性能问题 | 用户体验差 | 优化数据结构，懒加载 |
